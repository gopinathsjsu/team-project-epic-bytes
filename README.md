## Feature Set

### Admin Features
1. Login
2. Add hotels
3. Update/Delete hotels
4. Add rooms
5. Add amenities
6. View all bookings
7. Edit/Delete bookings for the customer.

### User Features
1. Signup / Signin
2. Search hotel based on location
3. Create bookings (filters: location, number of guests, number of rooms)
4. Select amenities for each room
5. View bookings
6. Update bookings
7. Delete bookings
8. Manage rewards
